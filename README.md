# Market Intelligence Pipeline

## Objective

Design and implement an end-to-end market intelligence pipeline using official
UK Companies House data. The system ingests structured company data via API,
validates and normalises it, stores it in Microsoft SQL Server, and produces
analytics-ready outputs to support commercial and financial analysis of UK SMEs.


## Target Universe
UK SMEs (to be defined by industry and region).

## Data Sources
Public business directories and company websites.

## Outputs
- Normalised SQL database
- Ranked company target list
- Recurring analytical summary report

## Tools
Python, Pandas, SQLAlchemy, SQL, Git
