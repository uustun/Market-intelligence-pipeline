# Market-intelligence-pipeline

## Objective
Build an automated pipeline that ingests public UK company data, cleans and normalises it, stores it in SQL, and outputs a ranked target list and recurring analytical report to support commercial decision-making.

## Target Universe
UK SMEs (to be defined by industry and region).

## Data Sources
Public business directories and company websites.

## Outputs
- Normalised SQL database
- Ranked company target list
- Recurring analytical summary report

## Tools
Python, Pandas, SQLAlchemy, SQL, Git
